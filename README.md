#SimpleSales SaaS 

Projeto SaaS simples para controle de vendas, desenvolvido como projeto de portfolio. 

## Objetivo 
Aprender na pratica o desenvolvimento de um Saas utilizando FastAPI, banco de dados e boas praticas 

## Funcionalidades (MVP) 
- Autenticacao simples 
- Cadastro de clientes 
- Cadastro de vendas 
- Relatorios de vendas por periodo 

## Stack 
- Python 
- FastAPI 
- SQLite 
- SQLAlchenmy
